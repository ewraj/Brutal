# Brutal

BRUTAL is a zero-sycophancy AI technical advisor designed for ruthless efficiency and radical candor. This project has evolved into a robust web application that provides a premium, dark-mode interface for interacting with the Brutal AI persona.

## Features

- Puter.js Integration: Completely free, unlimited Gemini access directly in the browser via Puter.js.
- Multiple Personas: Switch between default Brutal mode, Code Review, Debate Mode, Essay Critic, and even Custom Personas.
- Persistent Chat History: All conversations are automatically saved, and you can pin important chats or search through past sessions.
- Attachments: Support for uploading and analyzing text files, code snippets, and images.
- Obsidian Export: One-click export of any conversation into Obsidian-compatible Markdown.
- Import Chats: Resume past sessions by importing previously exported Markdown files.
- Code Copy: Easily copy code blocks generated by the AI with a single click.
- Responsive Design: Fully functional on both desktop and mobile devices with a collapsing sidebar.
- Keyboard Shortcuts: Quickly create new chats or trigger exports.

## Access

The Brutal Web App is hosted live on GitHub Pages and can be accessed directly at: https://ewraj.github.io/Brutal/

Because the application is entirely client-side and relies on Puter.js for all backend services, no local installation or server is required to use it. Simply open the link and log in with your Puter account.

## Project Structure

- index.html: The main web interface structure.
- app.js: Core application logic, chat management, API integration, and rendering.
- style.css: The dark-mode styling system.
- brutalv0.4.py / brutal / persona: Legacy CLI tool files kept for historical reference or headless usage.

## The Brutal Philosophy

Brutal follows a strict operational baseline: zero-tolerance for inefficiency, no filler, and providing optimal solutions immediately. Every component of this web application is designed to adhere to those principles, keeping you focused entirely on the code and logic.
