# Brutal

BRUTAL is a zero-sycophancy AI technical advisor designed for ruthless efficiency and radical candor.

## Components

### 1. Terminal Wrapper (`brutalv0.4.py`)
A CLI tool powered by Puter.js that provides free, unlimited Gemini access directly from your terminal. It features a custom "Brutal" persona and a local bridge server to interact with the AI.

**Usage:**
```bash
./brutal "How do I optimize this loop?"
```
or enter interactive mode:
```bash
./brutal
```

### 2. Web Application (`index.html`)
A premium, dark-mode web interface for the Brutal AI, featuring real-time markdown rendering and persistent sessions.

## Project Structure
- `brutalv0.4.py`: Latest CLI version.
- `brutal`: Launcher script.
- `persona`: AI behavioral instructions.
- `index.html`, `style.css`, `app.js`: Web application files.
- `LICENSE`: MIT License.

## Persona
Brutal follows a strict operational baseline: zero-tolerance for inefficiency, no filler, and providing optimal solutions immediately.
